## ✈️ About Travelscapes

Travelscapes is a responsive travel and tour booking website developed to provide users with a smooth travel planning and hotel booking experience. The platform allows users to explore destinations, view hotels, apply filters, and make bookings through an interactive and user-friendly interface.

---

## 🚀 Features

- Responsive modern UI
- Hotel and destination browsing
- Hotel filtering by cost and ratings
- Booking functionality
- Admin login system
- MySQL database integration
- User-friendly navigation
- Mobile-friendly design

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- PHP
- MySQL
- Bootstrap Icons
